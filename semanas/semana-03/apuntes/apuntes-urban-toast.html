📌 El Ritual Semántico

Hoy hemos transformado un borrador plano en una estructura web profesional. No se trata solo de que "se vea bien", sino de que el navegador entienda qué es cada cosa.

  • <main>
  • <header>
  • <footer>
  • <address>
  • <video>
🌉 La Metáfora del Puente: En la Iteración 6 aprendimos que las etiquetas <a> son los puentes que conectan nuestras islas de información. Sin el atributo href, el puente está roto; con #, el puente está en construcción pero es sólido.

🎬 Dominando lo Multimedia

La implementación del vídeo fue el reto estrella. Aprendimos que para una experiencia de usuario fluida (UX), debemos controlar tres factores:

  1. Controles: Dar el poder al usuario.
  2. Autoplay + Muted: La llave maestra para saltar las restricciones de los navegadores modernos.
  3. Poster: El "maquillaje" previo para que la web nunca se vea vacía.

⚠️ Troubleshooting: ¿Por qué el vídeo se quedó "mudo"?

Descubrimos que no todo es culpa del código. El primer intento falló por Hotlinking (bloqueo de servidor externo) y por las políticas de seguridad de Chrome que impiden vídeos automáticos con sonido.
Solución: Servidor de confianza + Atributo muted.
